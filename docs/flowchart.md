## 六、关键决策点对比

### 改进前 vs 改进后

| 决策点 | 改进前 | 改进后 |
|--------|--------|--------|
| 作者匹配 | 全名匹配 或 姓氏匹配 | 多作者拆分 + 顺序交换 + 简写处理 |
| 标题匹配 | 子串 + 段落 | 增加系列名称剥离 |
| 阈值 | 50分 | 40分 |
| 兜底 | LLM被注释 | 启用LLM API调用 |

### 多作者匹配示例

```
改进前:
  搜索: "David Jones; Jinli He (Editors)"
  结果: "David Jones"
  匹配: ✗ 失败 (normalize后整体不包含)

改进后:
  搜索: "David Jones; Jinli He (Editors)"
  拆分: ["David Jones", "Jinli He"]
  结果: "David Jones"
  匹配: ✓ 成功 (第一个作者匹配)
```

### 姓名顺序匹配示例

```
改进前:
  搜索: "Chenyang Li"
  结果: "Li, Chenyang"
  匹配: ✓ 成功 (姓氏Li匹配)

改进后:
  搜索: "Chenyang Li"
  结果: "Li, Chenyang"
  匹配: ✓ 成功 (反序匹配，更可靠)
```
